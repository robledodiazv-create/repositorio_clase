## Descripción
Este proyecto fue desarrollado como práctica para aprender el uso de Git y GitHub desde terminal.

El objetivo es comprender cómo inicializar un repositorio, registrar cambios mediante commits, trabajar con ramas y subir proyectos a GitHub para colaboración en equipo.

## Tecnologías usadas
- Lenguaje C
- Git
- GitHub
